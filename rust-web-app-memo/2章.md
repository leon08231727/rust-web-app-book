# 2章環境構築
   ## 躓いた箇所
   ### 2.1.1 Rustツールチェインのインストール
   - ubuntuでRustツールチェインをインストールしたがコマンド入力してもエラーとなった
     → ubuntu再起動で解決
   - Linux上での環境構築が前提 <br>
   参考リンク:
       - https://learn.microsoft.com/ja-jp/windows/wsl/install
       - https://qiita.com/yutobomberair/items/903d91b17df00d8d9476 <br>
       リンクの内容を参考にVS CodeからリモートでLinux環境を触れるようにした
   ### 接続手順
   1. PowerShellで下記を実行してLinux環境を起動      
   ```powershell
   wsl.exe -d Ubuntu
   ```
   2. 起動したubuntuターミナル上で下記を実行
   ```bash
   cd ~/project-name code .
   ```
